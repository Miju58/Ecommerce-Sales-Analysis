# E-Commerce Sales Analysis Dashboard

## Project Overview
This project focuses on analyzing e-commerce sales data using SQL and Power BI.  
The dashboard provides insights into sales performance, customer behavior, regional sales distribution, and category-wise analysis.

---

## Tools & Technologies Used
- SQL (MySQL)
- Power BI
- Excel / CSV
- Data Analysis
- Data Visualization

---

## Dataset
The dataset contains:
- Order details
- Customer information
- Product categories
- Regional sales data
- Monthly sales records

---

## Key KPIs
- Total Sales
- Total Orders
- Average Sales
- Regional Sales Analysis
- Category-wise Sales
- Customer Revenue Analysis

---

## Dashboard Features
- Interactive slicers for Region, Category, and Segment
- Monthly sales trend analysis
- Sales by region visualization
- Customer revenue insights
- Category-wise sales distribution
- State-wise sales map

---

## Project Highlights
- West region generated the highest sales.
- Technology category contributed the highest revenue.
- November recorded the highest monthly sales.
